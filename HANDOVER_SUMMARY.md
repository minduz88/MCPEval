## 🎯 **핵심 혁신 포인트**

### **다중 서버 크로스 도메인 시나리오**
```
기존: 단일 서버 → 단조로운 시나리오
혁신: 다중 서버 → 크로스 도메인 복합 시나리오

예시:
"삼성전자 분석 + 뉴스 감성 + 투자계좌 확인"
= 3개 서버 협력 시나리오
```

### **짬통 에이전트 패턴 재현**
- 실제 JT Agent 동작을 MCPEval로 완벽 모방
- 도메인 간 넘나드는 복합적 금융 서비스 시나리오
- 현실적이고 다양한 사용 패턴 학습 데이터 생성

1. 원본 HMA Gateway 코드 직접 복사
